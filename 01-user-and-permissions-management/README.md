# 01 - User and Permissions Management

This folder contains a Bash practice script focused on basic Linux filesystem operations and permission management.

## File

- `user_permissions_lab.sh`

## What This Script Practices

- creating directories with `mkdir -p`
- navigating paths
- creating files with redirection
- copying files with `cp` and `cp -i`
- displaying file contents with `cat`
- changing permissions with `chmod`
- appending command output to a file
- wildcard matching with `ls ?ata*`
- removing files with `rm`
- removing directories with `rmdir`
- creating a secure directory using octal permissions

## Main Concepts

This script demonstrates:

- Linux path handling
- file and directory management
- symbolic and octal permission concepts
- safe overwrite behavior
- shell prompt customization
- cleanup workflows

## Run

From the repo root:

```bash
chmod +x 01-user-and-permissions-management/user_permissions_lab.sh
./01-user-and-permissions-management/user_permissions_lab.sh
