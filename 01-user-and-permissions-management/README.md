# Linux Practice Scripts 

This repo contains small Linux and Bash practice scripts organized by topic and difficulty over time.

## Structure 
- '01-user-and-permissions-management/'
- 'user_and_permissions_management.sh'

### '01-user-and-permissions-management/user_and_permissions_management.sh'

This script practices foundational Linux command-line skills, including:

- creating nested directories
- working with absolute paths 
- creating and copying files 
- viewing file contents 
- changing file permissions 
- appending output to files 
- interactive overwrite protection with 'cp-i'
- wildcard pattern matching
- temporarily customizing the shell prompt
- removing files and directories
- setting secure directory permissions using octal notation

## Skills Demonstrated 

- Bash scripting
- file and directory management 
- Linux permissions
- path navigation 
- shell redirection
- wildcard expansion
- cleanup operations
- secure directory setup

## How to run

From the repo root:

'''bash
chmod +x 01-user-and-permissions-management/user-and-permissions-management.sh
./01-user-and-permissions-management/user-and-permissions-management.sh
